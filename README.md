# C. M. Conlan Contractors & Builders Website

## Files Included

This package contains all the files needed for your website:

### Main Files
- **index.html** - Main website file with updated hero section
- **staticwebapp_config.json** - Azure Static Web App configuration

### Images
- **conlanlogo.png** - Company logo
- **chrisconlanheadshot.png** - Chris Conlan headshot
- **home_1.jpg through home_7.jpg** - Home portfolio images

## Deployment Instructions

1. All files should be in the same directory
2. Upload all files to your web hosting or Azure Static Web App
3. The images will automatically load in the slideshow and throughout the site

## Features

- Responsive design that works on all devices
- Automatic slideshow with manual navigation controls
- Smooth scrolling navigation
- Mobile-friendly menu
- Professional gallery showcasing your work

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
